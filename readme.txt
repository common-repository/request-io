This plugin helps you add Request.io to your WordPress website, an easy way to improve the page speed of your website. Requires an active, paid subscription which can be registered on [Request.io](http://www.request.io).

== Description ==
This plugin helps you configure your WordPress website to use Request.io. 

Add Request.io to any site and improve page speed today. No need to switch systems or invest in lengthy development projects.

= Improved customer experience =
Every step of the customer journey requires the loading of content. Delight your customers with faster load times.

= Higher return on advertising spend =
A substantial part of advertising budgets are lost due to impatience. Keep customers by not having them wait.

= Increased profitability =
Helping your customers reach their goals faster directly translates into lower acquisition costs and higher revenue.

= Notes =
This plugin communicates with the Request.io optimization engine in order to optimize your Wordpress site. Read more at [www.request.io](http://www.request.io).

== Installation ==
Sign up at [Request.io](http://www.request.io).

1. Upload the Request.io WordPress plugin to your blog
2. Activate the plugin through the Request.io menu in WordPress.
3. Enter your Request.io API-key in the plugin's settings page, then save.

You're ready to start using Request.io!
